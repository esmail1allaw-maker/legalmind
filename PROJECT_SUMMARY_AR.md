## 🔄 التدفق الكامل للبيانات

### عند إضافة عميل:
```
1. المستخدم يفتح نموذج إضافة عميل
2. يملأ البيانات (الاسم، الهاتف، البريد)
3. يضغط "حفظ الموكل"
4. التحقق من الصحة (client-side)
5. تعطيل جميع الحقول والزرارع
6. إرسال البيانات إلى Supabase
7. Supabase تُولد ID فريد وتحفظ البيانات
8. استقبال البيانات المحفوظة مع ID
9. تحديث State العملاء فوراً
10. إغلاق النموذج
11. عرض رسالة النجاح
12. العميل الجديد يظهر في القائمة
```

### عند إضافة قضية:
```
1. المستخدم يفتح نموذج إضافة قضية
2. يختار عميل من القائمة المنسدلة
3. يملأ بيانات القضية
4. يضغط "حفظ القضية"
5. التحقق من الصحة
6. البحث عن اسم العميل المربوط
7. تعطيل الحقول والزرار
8. إرسال البيانات (مع clientName) إلى Supabase
9. Supabase تحفظ القضية
10. تحديث State القضايا
11. تحديث عدد القضايا للعميل (+1)
12. إغلاق النموذج
13. عرض رسالة النجاح
14. القضية الجديدة تظهر في الكروت
15. عدد القضايا للعميل يزيد
```
